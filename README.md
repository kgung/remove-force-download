# Remove forced downloads

Inspired by [no-force-download](https://github.com/straws11/no-force-download/tree/main)

This extention removes the ?forcedownload=1 parameter from all links opened, thereby making browsing the web smoother.


# Usage
Requires npm
```bash
npx tsc
```
Then insert it into your favorite browser. (I'm using orion, but firefox should work fine)
